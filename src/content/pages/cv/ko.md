**2025.07 ~ 2026.06** · Java · Spring Boot · AWS · Docker · GitHub Actions · PostgreSQL

- **성능 엔지니어링·부하 테스트** — Prometheus·Micrometer·Zipkin으로 계측한 단계별 k6 부하 테스트에서, HikariCP 커넥션 풀을 10→20으로 두 배 늘려도 처리량이 거의 개선되지 않아 병목이 DB가 아님을 확인. - **CI/CD 파이프라인 설계** — GitHub Actions 기반 CI와 GitHub Secrets·Docker·Docker Hub·AWS 기반 CD를 아우르는 파이프라인을 설계·구축.
- **엔지니어링 프로세스 최적화** — 헥사고날 아키텍처 내에서 Use Case 설계 시 UML 시퀀스 다이어그램 작성을 의무화하는 사전 승인 체계를 도입해 팀의 기술적 방향을 정렬하고, PR 리뷰 시간과 머지 충돌을 크게 감소.
