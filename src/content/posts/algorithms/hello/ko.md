## 반갑습니다 👋

이 글은 블로그에서 자유롭게 쓸 수 있는 기능들을 보여주는 **예시**입니다.
이모지 😀, **굵게**, *기울임*, 그리고 `인라인 코드` 모두 자연스럽게 쓸 수 있어요.

> 💡 오른쪽 위 🌐 버튼을 누르면 이 글의 영어 버전으로 바로 전환됩니다. 주소는 바뀌지 않아요.

## 코드 스니펫

문법 하이라이팅과 복사 버튼이 기본으로 붙습니다.

```typescript
function greet(name: string): string {
  return `안녕하세요, ${name}님!`;
}

console.log(greet('raewoo0908'));
```

인라인으로도 `const x = 42;` 처럼 코드를 넣을 수 있습니다.

## 이미지

![샘플 이미지](/images/sample.svg)

이미지는 `public/` 폴더에 넣고 절대 경로로 참조하거나, 글 폴더에 함께 두고 상대 경로로 참조할 수 있습니다.

## 마무리

이제 새 글을 쓰려면 `src/content/posts/<카테고리>/<글이름>/` 폴더를 만들고
`ko.md`와 `en.md`를 넣으면 됩니다. 폴더만 추가하면 상단 메뉴에 카테고리가 자동으로 나타납니다. 🎉
